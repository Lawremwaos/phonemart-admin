1. Go to: https://github.com/new
2. Repository name: `phonemart-admin` (or any name you like)
3. Description (optional): "PHONEMART Admin Management System"
4. Choose: **Public** or **Private** (your choice)

7. **IMPORTANT:** Do NOT choose a license
8. Click the green **"Create repository"** button
